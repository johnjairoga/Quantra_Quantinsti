Folder Structure and How to Run Code Files

 

 


In this course, you have learned Python data structure, pre-processing of the financial data, various trading strategies based on the technical indicators, and analyse the performance of the strategy.

 

In this folder, you will find the data files, code files and necessary information needed to run the code files successfully.
